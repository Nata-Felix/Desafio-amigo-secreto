SORTEADOR DE AMIGO SECRETO!

Como funciona o sorteador de nomes?

O sorteador utiliza um algoritmo de seleção aleatória
para sortear dentro de uma lista criada pelo usuário.

Posso remover o nome de quem já foi sorteado?

Sim, você pode marcar a opção de remover sorteado para
retirar o nome da lista do sorteio!
